
# Boblancer Exchange Engine

Zipmex code challenge

## Installation

Run with docker

```bash
  docker build -t boblancer-zipmex .
  docker run boblancer-zipmex:latest
```

## Library **(npm)**

 - Mocha **(Testing)**
 - sorted-btree **(B+ tree for internal storage)**
